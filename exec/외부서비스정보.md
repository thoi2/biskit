## 2.3 OpenAI API

### **서비스 종류**

### **가입 및 설정**
1. **OpenAI Platform** 접속: https://platform.openai.com/
2. **계정 생성** 및 결제 정보 등록

### **필요 정보**

- **토큰 제한**: 500 tokens per request
